Armstrong Pull-up Tracker (v3) - One Page Per Day + Day 5 Picker

What's new:
- One page per day (tap Day 1–Day 5 at the bottom).
- Day 5 has a selector to repeat Day 1–4, with its own separate inputs and total.
- Autosave + offline still included.

Update your GitHub Pages repo:
1) Download this zip and unzip it.
2) Upload/replace these files in your repo root:
   - index.html
   - manifest.json
   - sw.js
   - icon-192.png
   - icon-512.png

If you don't see updates:
- Open the site in Chrome -> (⋮) -> Settings -> Site settings -> Storage -> Clear
- Then reopen. You may need to remove/re-add the installed app for the UI to refresh quickly.
